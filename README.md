# 똥피하기 💩 (Poop Dodge)

간단한 HTML5 캔버스 게임입니다. GitHub Pages로 바로 호스팅할 수 있습니다.

## 조작법
- PC: ← → 또는 A/D / 마우스 드래그
- 모바일: 하단 좌/우 버튼

## 배포 (GitHub Pages)
- 기본: GitHub Actions로 main 브랜치 푸시 시 자동 배포
- 주소 형식: https://<USERNAME>.github.io/<REPO>/

### 수동 설정(필요시)
- 저장소 Settings → Pages → Source: GitHub Actions
